To Use the Prezi:

1. Choose a course from the home view

2. Choose a course learning outcome from that course

3. Click the fully written out course learning outcome

4. Choose a KSA Bin from the list of bins

5. On the Google Sheet document, use the tabs at the bottom
	to navigate to the KSA Bin you chose from the Prezi

6. Observe how individual KSA relates to the course
